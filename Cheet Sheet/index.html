Different Languages

The 3 Honchos

There are three main languages that work together to help build websites, each have a different function and usability.

  • HTML
  • CSS
  • Javascript

HTML

Firstly let's learn about HTML and what it does, HTML stands for HyperText Markup Language it "is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript." 1

CSS

CSS also known as Cascading Style Sheets "is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript." 2

JavaScript

JavaScript is a high-level programming language, it is used by the vast majority to help incorporate third-party libraries. " All major web browsers have a dedicated JavaScript engine to execute the code on users' devices." 3

Layout

The start of every HTML document must look like this, it allows for browsers to read what is going on. This is how it will look broken up into section.

Boilerplate:

1 <!DOCTYPE html>
2 <html lang="en">
3 <head>
4 <meta charset="UTF-8">
5 <meta http-equiv="X-UA-Compatible" content="IE=edge">
6 <meta name="viewport" content="width=device-width, initial-scale=1.0">
7 <title> (What you want to appear in the browser) <title>
8 <link rel="stylesheet" href=".resources/css1/index.css">
9 </head>

The body can be broken up into 3 main sections:

  1. header
  2. main
  3. footer

All of these sections are embedded within the <body> elements but are created to help make everything look more readable.

  1. Header:

10 <body>
11 <header>
12 <nav>
13 <ul or ol> (if a list is required for navigation)
14 <li> (can add as many as needed) </li>
15 </ul or /ol>
16 </nav>
17 <img src="/" (if adding an image to the header)>
18 </header>

The<body> element isn't shut yet because the other two parts will continue within it, as children. If you want to learn how to make the navigation buttons clickable click here to read how.

  1. Main:

19 <main>
20 <article>
21 <div>
22 <h1>(Title)<h1>
23 <p>(Content will go here.)<p>
24 </div>
25 </article>
26 </main>

You can add as many <article> and <div> as needed. This will help to break the code into pieces and make it easier to read. The more content that is added, the more <div> you will have.

Command Lines

Tag Name Description
pwd Print Working Order Outputs the name of the current working directory
ls List Lists all files and directories in the working directory.
mkdir Make Directory Creates a new directory in the working directory.
cd Change Directory Switches you into the directory you specify.
touch Create New File Creates a new file in the working directory with the name provided.
